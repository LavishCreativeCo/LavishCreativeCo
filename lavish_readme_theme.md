## 👩🏾‍💻 About the Project

**Lavish Analytics Engine** is a machine learning pipeline designed to analyze **customer sentiment from social media comments, product reviews, and brand feedback.**

The goal is to help brands — especially **beauty, fashion, and digital businesses** — understand how their audience feels in real time.

Instead of guessing what customers think, brands can use **AI‑driven insights** to improve products, marketing, and engagement.

---

## 🚀 What This Project Does

This system:

• Collects customer feedback  
• Cleans and processes text data  
• Uses machine learning to classify sentiment  
• Generates insights brands can use to improve engagement  

The model predicts:

- Positive sentiment  
- Neutral sentiment  
- Negative sentiment  

---

## 💡 Real‑World Use Cases

This system could be used by companies like:

- Beauty brands analyzing product reviews  
- Fashion brands monitoring campaign reactions
